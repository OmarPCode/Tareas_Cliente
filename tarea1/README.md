# Tarea 1

## Descripción
Aplicación web de CV en línea que consume datos desde un JSON y utiliza **HTML**, **Sass**.  
Incluye un pipeline con **Gulp** para compilar estilos, scripts y copiar los assets a la carpeta `/dist`, con servidor local en el puerto **3000**.

## Instalación
```bash
cd tarea1
npm install
```
## Ejecución
- Es el Puerto 3000
- Ejecutar: 
```bash
npm run dev
```
## Cambiar correo del formulario
- El correo se configura en src/data/cv.json dentro de la sección contact.email.
- Ejemplo:
```bash
"contact": {
  "email": "omardponce@gmail.com"
}
```
- Ese correo será el destino del formulario al enviarse mediante FormSubmit
