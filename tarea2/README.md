# Tarea 2 — Comunicación entre Componentes Siblings en Angular

## Descripción
Este proyecto es una aplicación en **Angular** que demuestra la comunicación entre componentes **hermanos (siblings)** a través de un componente **padre**.  
La aplicación muestra una lista de títulos (libros, películas o canciones) en el panel izquierdo.  
Al seleccionar un elemento, se muestra su detalle en el panel derecho junto con un botón para **limpiar la selección**.  
Mientras no haya selección, se muestra un mensaje de aviso.

---

## Tecnologías utilizadas
- [Angular 17](https://angular.dev/) (standalone components)  
- TypeScript  
- HTML5  
- CSS3
